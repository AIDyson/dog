# PawRank - 狗狗颜值评分与排行榜

一个现代化的狗狗颜值评分网站，具备完整的排名系统和展示功能。

## 功能特性

### 1. 排行榜系统
- **全球排名**：展示全球所有狗狗的排名
- **本周Top10**：本周热门狗狗排名
- **本国排名**：按国家分类的排名
- 每个排名都显示：狗狗照片、分数（百分制）、投票数、动量值
- 交互式卡片，鼠标悬停有浮动和边框效果
- 点击可切换排名类型，查看详情跳转新页面

### 2. 狗狗主页
- 展示狗狗的详细信息
- 显示评分、排名、统计数据
- 主人故事和标签
- 评分趋势图表
- 社交分享功能

### 3. 上传流程
- **步骤1**：上传狗狗照片（支持拖拽）
- **步骤2**：付费解锁（$1.90/次）
- **步骤3**：填写狗狗信息（名称、年龄、故事）
- **步骤4**：获取评分和排名结果
- 生成分享海报

### 4. 最新上传展示
- 网格布局展示最新上传的狗狗
- 显示照片、名称、品种、产地、评分
- 标签系统
- 直接评分按钮

### 5. 3D地球
- 可旋转的3D地球模型
- 显示30个热门养宠国家
- 点击国家进入该国排名页面
- 悬停显示国家信息

### 6. 国家排名页
- 展示特定国家的所有狗狗排名
- 显示本国排名和全球排名
- 国家统计数据

## 技术栈

- **React 19** - 前端框架
- **React Router Dom** - 路由管理
- **Three.js & React Three Fiber** - 3D地球渲染
- **Framer Motion** - 动画效果
- **Lucide React** - 图标库

## 设计特点

- 深色主题设计
- 金色渐变强调色（#FFD700）
- 现代化的玻璃态效果（Glassmorphism）
- 流畅的过渡动画
- 响应式布局（PC优先）

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 访问规则

- **访客**：可以免费浏览所有排名和狗狗主页
- **上传用户**：需要付费（$1.90/次）才能解锁评分并参与排名
- 未付费的上传只能私有化查看

## 评分系统

当前评分系统为随机生成（75-98分），后续可接入AI训练的标准规则进行评分。

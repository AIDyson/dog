## 🚀 如何运行

### 1. 启动开发服务器
```bash
cd client
npm start
```

服务器会自动在 http://localhost:3000 启动

### 2. 访问网站
- **首页**: http://localhost:3000
- **排名列表**: http://localhost:3000/ranking/weekly
- **狗狗主页**: http://localhost:3000/dog/1
- **上传流程**: http://localhost:3000/upload
- **国家排名**: http://localhost:3000/country/Japan

## 🎯 核心功能说明

### 排名系统
- **评分规则**：百分制（当前为随机75-98分）
- **排名计算**：综合评分、投票数、动量值
- **动量值**：7天内的排名变化趋势
- **三种排名**：全球、本周、本国

### 上传流程
1. **选择照片** → 拖拽或点击上传
2. **付费解锁** → $1.90/次
3. **填写信息** → 名称、年龄、故事
4. **获得评分** → 随机生成评分和排名
5. **分享结果** → 生成海报，分享社交媒体

### 3D地球
- 使用 Three.js 渲染
- 30个国家标点
- 可旋转、缩放
- 点击进入国家排名页
